# Student-Productivity-Dataset-Analysis
This project analyses student lifestyle and study behavior using Excel data Analytics Technique
Dataset Variables include:
1. Age, Gender, Academic Level
2. Study Hours , Self Study Hours , Online Class hours , Gaming Hours
3. Social Media hours and Sleep hours
   
Analysis Goals:

     * Identify relationship between lifestyle habits annd academic productivity
     * Evaluate the impact of screen time and social media usage and sleep and Concentration.
     * Comapre study patterns across different education levels.

Methods Used:

1. Data Cleaning and preprocessing
2. Using Power Query to Better Cleaning and Create Conditional Column for Analysis
3. Categorization of Study Patterns
4. Use Pivot Table for Comparing Cleaned Data
5. Conditional formating for visual insights
6. Using Charts and Bars for Visual Analysis and Slicers for Particular insights.
7. Data bars and indicators for behavioral analysis

Key Findings:

1. Excessive screen time is associated with lower Sleep Quality.
2. Higher self-study hours generally improve concentration levels.
3. Social media addiction correlates with reduced productivity.

Tools:

1. Excel
2. Data Visualization
3. Descriptive Analysis.
